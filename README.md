# Motor3D

Motor 3D desarrollado completamente desde cero en JavaScript.

## ¿Por qué?

- Entender el 3D
- Tener una base reutilizable para otros proyectos
- Aprender geometría computacional

## Estructura del repositorio

- Proceso/
Contiene las distintas etapas del desarrollo.

- Motor/
Contiene la versión más reciente del motor.
